# QuickHub Popup - Premium Shortcut Manager

QuickHub is a modern, glassmorphic dashboard that lives in your Chrome toolbar. Access your favorite sites instantly without cluttering your New Tab page.

## 🚀 Key Features
- **Toolbar Access**: Click the extension icon to see your shortcuts in a clean popup.
- **Unlimited Shortcuts**: Add your favorite sites with automatic favicon fetching.
- **Drag & Drop**: Reorder your shortcuts exactly how you want them.
- **Categories**: Group your links (Social, Dev, Media, etc.).
- **Deep Customization**: Change theme colors and dark/light modes.
- **Import/Export**: Backup your data as a JSON file.
- **Premium Design**: Built with glassmorphism and smooth animations.

## 🛠 Installation Instructions (Unpacked Extension)

1. **Open Chrome** and navigate to `chrome://extensions/`.
2. **Enable Developer Mode** by toggling the switch in the top-right corner.
3. Click the **"Load unpacked"** button.
4. Select the folder where you saved these files
5. Click the **Puzzle Piece icon** in your toolbar and pin **QuickHub** for easy access!

## ⌨ Keyboard Shortcuts
- Press `/` to focus the search bar when the popup is open.

## 📂 Project Structure
- `manifest.json`: Extension configuration (Popup mode).
- `popup.html`: Main popup layout.
- `styles.css`: Glassmorphic styling.
- `script.js`: State management and logic.
- `sortable.min.js`: Local drag & drop library.
- `assets/`: Extension icons.
